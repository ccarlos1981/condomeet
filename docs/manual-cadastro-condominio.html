Capítulo 3

Nome e Endereço

Preencha os dados do condomínio

📝 Formulário de Dados

1

Nome do Condomínio

Informe o nome oficial. Ex: "Residencial Real Park", "Edifício Aurora".

2

Endereço Completo

CEP, rua, número, bairro, cidade e estado. O CEP preenche automaticamente os demais campos.

3

CNPJ (Opcional)

Se o condomínio possui CNPJ, informe aqui para facilitar futuras integrações.

Formulário — Nome e Endereço do Condomínio
✅
Dica: Use o nome oficial do condomínio como aparece nos documentos. Os moradores irão pesquisar por esse nome ao se cadastrar.

Capítulo 5

Configurar Estrutura

Cadastre blocos, torres e apartamentos

🏗️ Por que configurar a estrutura?

A estrutura permite que o sistema identifique cada unidade do condomínio. Isso é essencial para:

  • Vincular moradores às unidades corretas
  • Direcionar encomendas ao apartamento certo
  • Organizar avisos por bloco ou andar
  • Controle de visitantes por unidade

📋 Passo 1: Cadastrar Blocos / Torres

1

Acesse "Estrutura" no Painel Admin

No painel administrativo, toque em Estrutura do Condomínio.

2

Adicione um Bloco

Toque no botão "+" para adicionar um novo bloco. Ex: "Bloco A", "Torre Norte".

Passo 1 — Cadastrar Blocos

📋 Passo 2: Cadastrar Apartamentos

1

Selecione o Bloco

Toque no bloco desejado para abrir a lista de unidades.

2

Adicione os Apartamentos

Cadastre as unidades individualmente ou use a geração automática.

Cadastrar Apartamentos
Gerar Unidades em Lote
🚀
Dica Pro: Use a opção "Gerar Unidades" para cadastrar todos os apartamentos de uma vez! Basta informar o intervalo (ex: 101 a 404) e o sistema cria automaticamente.

Cadastre os espaços que os moradores poderão reservar pelo app. Exemplos comuns:

Acesse "Configurar Reservas"

3

Preencha os Dados

Lista de Áreas Comuns
Formulário — Nova Área Comum

Capítulo 7

Configurar Acesso

Controle quais funções cada perfil pode ver no app

👥 Por que configurar?

Cada condomínio tem necessidades diferentes. A tela de Configurar Menu permite que você personalize quais funcionalidades ficam visíveis para cada tipo de perfil:

🏠 Morador
🔑 Porteiro
👔 Síndico
🧹 Zelador

📋 Como Funciona

1

Acesse "Configurar Menu"

No painel admin, na seção Parametrização, toque em Configurar Menu.

2

Selecione a Função

Use o dropdown para navegar entre funções (ex: Reservas, Documentos, Encomendas).

3

Marque os Perfis

Ative/desative a visibilidade para cada perfil. Ex: Reservas → apenas Morador e Síndico.

4

Salve

Toque em Salvar no topo da tela. A configuração é aplicada imediatamente.

Configurar Menu — Acesso por Perfil
✅
Dica: A aba "Ordem" permite reorganizar a posição dos botões no menu do app. Menor número = aparece primeiro.
